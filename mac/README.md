# Automated mac setup

You can either run all the scripts together or run them individually for whatever you need.

All:

```zsh
$ cd dotfiles/mac
$ source all.sh
```

Individual:

```zsh
$ cd dotfiles/mac
$ source setup.sh
$ source brew.sh
$ source terminal.sh
$ source starship.sh
```
